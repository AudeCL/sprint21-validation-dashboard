How to read this table

The Solutioning Status indicates the maturity of each User Story and whether it is ready for stakeholder review.

🔴 In Progress The Salesforce team is actively working on the User Story or is awaiting additional inputs before it can be completed. If validation is required, the item is not yet ready for review. Validation will be requested once the solution reaches a more stable state.
🟡 Under Finalisation The Salesforce team is finalising the User Story. If validation is required, reviewers may already start providing feedback, understanding that minor updates or clarifications may still be introduced before the User Story is finalized.
🟢 Finalised The Salesforce team has completed the User Story. If validation is required, the item is ready for stakeholder review and approval.
